Accounts

A modular user account manager for FileMaker with the following functions and features:

  • Create
  • Delete
  • enable/disable
  • reset password
  • change privilege set
  • configurable settings
  • email user a “welcome” email
  • basic UI for account management

The module is designed to work with single and multi-file solutions where the accounts and privilege sets are the same in every file.
